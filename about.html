ABOUT: Technologies and approach

The aim of this site is:

  • To share my father's experiences and to preserve his legacy.
  • Provide access to documents and photos that might be of interest to researchers.
  • Develop my technology and coding skills.

Summary of tools and technologies used:

HTML 5

CSS 3

GIT

Webstorm

Amazon S3

Fonts

Markup of the site was done in HTML5.

Styling was done using CSS3, primarily using the CSS Grid Layout.

Source is manged under my git account, using Atlassian SourceTree.

HTML and CSS coding was done using Jetbrains WebStorm IDE.

The Website is hosted on Amazon S3.

Images are stored on "S3" CDN.

URL (DNS) is registered on Amazon Route 53.

The font style comes from Google Fonts using the 'Special Elite' font.
