Markup of the site was done in HTML 5.

Styling was done using CSS 3, primarily using the CSS Grid Layout.

Source is manged under git, using Atlassian "SourceTree".

HTML and CSS coding was done using Jetbrains "WebStorm" IDE.

The Website is hosted on Amazon "S3".

Images are stored on "S3" CDN.

URL (DNS) is registered on Amazon "Route 53".

The font style comes from Google Fonts: 'Special Elite' font.

Other values of align-items are:

  • stretch
  • start
  • end
  • center
